VehicularMedia is a Portfolio website that allows users to contact or view the work and social media profiles of Vehicular Media

## Features

- **Conditional Rendering:** scales to mobile or desktop viewports
- **SMTP Enabled Contact Form:** uses AWS SES for the contact page to email owners of Vehicular Media
- **Animations:** animated using [Framer Motion](https://www.framer.com/motion/transition/) for an interactive user experience

![](/vehicular.png)

## Built with

- [Next.js](https://nextjs.org)
- [tailwindcss](https://tailwindcss.com)
- [AWS SES](https://aws.amazon.com/ses/)
- Deployed on [Vercel](https://vercel.com?utm_source=envshare)
